
1. Which of the following properties specifies the transparency of an element?
Answer: opacity

2. Which of the following options is used as a shorthand property to describe all transition-related properties?
Answer: transition

3. Which of the following options can be used to convert a vertical list into a horizontal list?
Answer: set the list items to display: inline; set the list items to display: inline-block;

4. Which property is used for the shadow-like effect for an element?
Answer: box-shadow

5. Which of the following values are required for the CSS box-shadow property?
Answer: h-offset, v-offset 

6. How to apply a style to every element of the html page?
Answer: * {}

7. There are different headings inside a HTML page with class 'h-style'. Which of the following is the best way to apply a CSS style only to immediate <span> child that is inside a <h2> 
tag whose class is 'h-style'? (Immediate child means that it is a immediate child node of the element in HTML DOM tree)
Answer: h2.h-style > span

8. Can we make our own custom attributes?
Answer: Yes

9. Which of the following represents the selection of only those <ul> that are immediately preceded by the <p> element, i.e. only the adjacent element?
Answer: p + ul { ..... }

10. How to apply style to only those <input> tags which have 'disabled' attribute to them?
Answer: input[disabled] { ..... }

11. Which of the following will apply style to only those <input> tags which have the value of their 'type' attribute as 'text' ?
Answer: input[type="text"] { ..... }

12. How to target all the images whose extension is 'jpg'?
Answer: img[src$=".jpg"] { ..... }

13. Which of the following is a correct example of a custom attribute?
Answer: data-file="image"

14. Select correct statements about CSS Box-shadow:
Answer: box-shadow property is used to produce a shadow-like effect for an element; Syntax for box-shadow is box-shadow : none | h-offset v-offset blur-radius spread-radius color; h-offset,
v-offset are required values while blur, color, spread are optional values.
