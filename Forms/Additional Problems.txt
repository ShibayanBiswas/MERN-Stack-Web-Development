
1. In HTML Audio/Video,which attribute sets or returns whether it should start playing as soon as it is loaded?
Answer: autoplay

2. Which of the following element defines a group of related options in a drop-down list?
Answer: <optgroup> [The optgroup tag is used to group related options in a dropdown list. This makes it easier to understand their choices when looking at a large list]

3. Which attribute will make a checkbox selected/ ticked by default?
Answer: checked

4. What is the value of rel attribute to describe a favicon?
Answer: icon [rel attribute defines the relationship between the current page and the linked resource. rel = “icon” value describes the favicon or web page icon that visually represents the website]

5. What is the role of charset attribute?
Answer: the character encoding for the HTML document

6. In which of the following ways can a submit button be created?
Answer: <input type=”submit” value=”Submit” />, <button>Submit</button>, <button type=”submit”>Submit</button> 

7. When form data contains sensitive information, then which method should be used?
Answer: Post [The post method is used to send critical data. The data is automatically passed in header of the request send and is accessible to the server]

8. What is the correct syntax to add user controllable mp4 video to the HTML?
Answer: <video controls><source src="xyz.mp4" type="video/mp4"></video>
